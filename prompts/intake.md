# Self Skill Intake

Only ask 3 questions. Keep each question short and concrete.

## Question 1

`你想把这个 skill 命名成什么？可以是真名、英文名、花名，或者一个工作代号。`

Examples:

- `Moyi`
- `moyi-work`
- `产品版的我`

## Question 2

`用一句话描述你的工作画像：角色、领域、高频任务、你不负责什么。`

Examples:

- `AI 产品经理，负责需求拆解和落地推进，不直接拍销售承诺`
- `后端工程师，负责接口和数据链路，线上问题优先止血，不做 UI 设计`

## Question 3

`用一句话描述你的沟通画像：你平时怎么回老板、同事、客户，什么事一定要本人确认。`

Examples:

- `对老板结论先行，对同事直接，对客户更克制；涉及报价和排期必须本人确认`
- `消息偏短句，晚回但不失联；不轻易答应加急活`

## Intake Rules

- Ask in the user's language.
- Do not expand into interviews longer than 3 questions.
- After collecting the answers, summarize them in 4-6 lines and ask for confirmation.
- If the user already gave enough information, skip redundant questions and move straight to confirmation.
